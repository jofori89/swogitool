Help?

Type a keyword to find the card(s) by name, ability, artist, ID.


You may also type logical expressions. Example:
= ((Stm + Def) > 8 && Def >= 2) || Atk >= 8

Which reads "Any card with the sum of it's defense and stamina greater then 8 and a minimum of 2 defense; or otherwise a card with at least 8 attack."


Premitted operations:

  • basic arithmetic +, -, /, *
  • basic logic operators ==, <=, >=, <, >
  • logical operations && (and), || (or), ! (not)
  • basic (loose) string comparison =~
  • string literals " (double quote) and ' (single quote).
  • paranthesis, use wisely ;)

Variables:

  • basic: Atk, Def, Stm, Cp, Name, Size, Id, Faction, Limit
  • contextual: hasAbility, isFlw, isSpell, isChar, isVita, isAcademy, isCrux, isDL, isSG
